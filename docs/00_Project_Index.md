# LubeLogger Project Index

## Daily Vehicle Inspection (Public Form)

This project adds a public-facing, mobile-friendly vehicle inspection form to LubeLogger. Drivers can scan a QR code to fill out inspections without needing a full account.

### Documented Components

- [[PublicController]] - Handles unauthenticated requests and submissions.
- [[InspectionView]] - The mobile-friendly inspection form.
- [[FinishedView]] - The success page after submission.
- [[InspectionRecordsUI]] - UI enhancements for QR code generation.
- [[InspectionScript]] - JavaScript logic for handling QR codes and modals.
- [[OptionalInspectionCost]] - Configuration to make cost optional in inspections.
