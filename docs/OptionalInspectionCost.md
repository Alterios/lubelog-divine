# Optional Inspection Cost

This feature allows users to configure whether the "Cost" field is required when adding or editing vehicle inspection records on a per-vehicle basis.

## Key Changes

- **Per-Vehicle Configuration**: A new toggle in the Vehicle Edit/Add modal allows users to specify if inspection costs are optional for that specific vehicle.
- **Frontend Validation**: The inspection record submission logic evaluates the vehicle's setting before enforcing the cost field requirement.
- **Backend Persistence**: The `Vehicle` model and data access layer now track this preference.

### Frontend Validation (`inspectionrecord.js`)

The `getAndValidateInspectionRecord` function was updated to conditionally validate the cost.

```javascript
if (GetVehicleId().inspectionCostOptional) {
    if (inspectionCost.trim() != '' && !isValidMoney(inspectionCost)) {
        hasError = true;
        $("#inspectionRecordCost").addClass("is-invalid");
    } else {
        $("#inspectionRecordCost").removeClass("is-invalid");
    }
} else {
    if (inspectionCost.trim() == '' || !isValidMoney(inspectionCost)) {
        hasError = true;
        $("#inspectionRecordCost").addClass("is-invalid");
    } else {
        $("#inspectionRecordCost").removeClass("is-invalid");
    }
}
```
